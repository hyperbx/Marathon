Picture Font

The Picture Font (PFT) format for Sonic '06 defines the placeholder icons for strings in the Text (MST) format.

After reading the BINA header, we end up at the first four important chunks of bytes.

00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00

  • This String is the signature of this BINA format.
  • This UInt32 is a pointer (plus 0x20) to the name of the texture.
  • This UInt32 is a pointer (plus 0x20) to the start of the null-terminated string table.
  • This UInt32 is a pointer (plus 0x20) to the start of the offset table.

After receiving this data, we can now start reading the offset table.

00 00 00 00 00 00 00 00 00 00 00 00 00 00 00 00

  • This UInt32 is a pointer (plus 0x20) to the name of the entry.
  • This UInt16 is the X co-ordinate of the current entry in the texture.
  • This UInt16 is the Y co-ordinate of the current entry in the texture.
  • This UInt16 is the Width of the current entry in the texture.
  • This UInt16 is the Height of the current entry in the texture.
  • Next entry in the offset table.
